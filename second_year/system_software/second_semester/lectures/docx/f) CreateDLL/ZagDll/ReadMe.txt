========================================================================
    БИБЛИОТЕКА MICROSOFT FOUNDATION CLASS. Обзор проекта 
    ZagDll 
========================================================================


Эта библиотека DLL ZagDll создана автоматически с помощью 
мастера приложений.  Эта библиотека DLL демонстрирует основные принципы 
использования классов Microsoft Foundation, и может использоваться в качестве 
основы для написания собственных библиотек DLL.

В этом файле представлена сводка содержимого всех файлов, входящих в состав 
библиотеки DLL ZagDll.

ZagDll.vcxproj
    Основной файл проекта VC++, автоматически создаваемый с помощью мастера 
    приложений.
    Он содержит данные о версии языка Visual C++, использованной для создания 
    файла, а также сведения о платформах, настройках и свойствах проекта, 
    выбранных с помощью мастера приложений.

ZagDll.vcxproj.filters
    Это файл фильтров для проектов VC++, созданный с помощью мастера 
    приложений. 
    Он содержит сведения о сопоставлениях между файлами в вашем проекте и 
    фильтрами. Эти сопоставления используются в среде IDE для группировки 
    файлов с одинаковыми расширениями в одном узле (например файлы ".cpp" 
    сопоставляются с фильтром "Исходные файлы").

ZagDll.h
    Это основной файл заголовков библиотеки DLL.  В этом файле
    содержится объявление класса CZagDllApp.

ZagDll.cpp
    Основной исходный файл библиотеки DLL.  В этом файле содержится класс 
    CZagDllApp.

ZagDll.rc
    Это список всех ресурсов Microsoft Windows,
    используемых программой.  В него включены значки, рисунки и курсоры,
    хранящиеся в подкаталоге RES.  Этот файл можно редактировать 
    непосредственно в Microsoft Visual C++.

res\ZagDll.rc2
    Этот файл содержит ресурсы, не редактируемые в Microsoft
    Visual C++.  Все ресурсы, не редактируемые редактором ресурсов,
    следует поместить в этот файл.

ZagDll.def
    В этом файле содержатся сведения о библиотеке DLL, необходимой
    для выполнения приложения в Microsoft Windows.  В нем определяются такие 
    параметры, как имя и описание библиотеки DLL.  Также в файле
    выполняется экспорт функций из библиотеки DLL.

/////////////////////////////////////////////////////////////////////////////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
    Эти файлы используются для построения файла предкомпилированного заголовка 
    (PCH) с именем ZagDll.pch и файла предкомпилированных типов 
    с именем StdAfx.obj.

Resource.h
    Это стандартный файл заголовков, в котором задаются новые идентификаторы 
    ресурсов.
    В среде Microsoft Visual C++ можно прочитать и обновить такой файл.
