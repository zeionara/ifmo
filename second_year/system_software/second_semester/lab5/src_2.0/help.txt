1. Критические секции - вывод информации о потоках, без них в консоли - анархия
2. События - синхронизация сортирующих потоков, без них - неверный результат сортировки
3. Мьютекс - синхронизация доступа к файлу, без него - очередная строка не сохраняется в файле
4. Семафор - ограничение числа потоков, одновременно выполняющих поиск в файле
